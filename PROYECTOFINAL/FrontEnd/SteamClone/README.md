# PlayVerse

Frontend de PlayVerse, una tienda academica de videojuegos creada con Angular.

Eslogan: Tu universo de videojuegos en un solo lugar.

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`.

## Building

To build the project run:

```bash
ng build
```
