# 💻 VMT Bootcamp - Full Stack Developer 2026

Repositorio donde se recopilan todos los ejercicios, prácticas y proyectos desarrollados durante la **Mentoría VMT Bootcamp Full Stack Developer**.

El objetivo de este repositorio es documentar el proceso de aprendizaje a través de diferentes tecnologías utilizadas tanto en el desarrollo Front-End como Back-End, bases de datos y un primer borrador del proyecto final.

---

Durante esta mentoría se trabajó en el aprendizaje de:

- Desarrollo Front-End
- Desarrollo Back-End
- Diseño y consumo de APIs REST
- Programación orientada a objetos
- Arquitectura MVC
- Bases de datos relacionales
- Git y GitHub
- Buenas prácticas de desarrollo
- Trabajo con proyectos Full Stack
